🏠 Interior Design API Node.js + Express

Backend reference for Portfolio, Inquiries & Blog — running on localhost:5000

Quick Start

npm install npm run dev    # development (auto-reload) npm start     # production
💡 Production tip The server currently uses in-memory storage. Replace the arrays with MongoDB (Mongoose) or PostgreSQL (Prisma) for persistent data.
🖼️

Portfolio Projects

▼
GET
/api/portfolio
Fetch all projects
GET
/api/portfolio/:id
Fetch single project by ID
POST
/api/portfolio
Create a new project (supports image upload)
form-data: title*, description, roomType, tags, images[]
PUT
/api/portfolio/:id
Update project details
JSON: { title, description, roomType, tags }
DELETE
/api/portfolio/:id
Delete a project
📬

Client Inquiries

▼
GET
/api/inquiries
Fetch all client inquiries (admin)
POST
/api/inquiries
Submit a contact form inquiry
JSON: { name*, email*, message*, phone, serviceType }
PATCH
/api/inquiries/:id/status
Update status: new → read → replied
JSON: { status: "read" | "replied" }
DELETE
/api/inquiries/:id
Delete an inquiry
📝

Blog Posts

▼
GET
/api/blogs
Fetch all blogs
query: ?category=trends &published=true
GET
/api/blogs/:id
Fetch by ID or slug
POST
/api/blogs
Create a blog post (with cover image)
form-data: title*, content*, category, tags, published, coverImage
PUT
/api/blogs/:id
Update a blog post
JSON: { title, content, category, tags, published }
DELETE
/api/blogs/:id
Delete a blog post
